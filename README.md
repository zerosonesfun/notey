# notey
A single HTML file that lets you save notes. [Demo](https://zerosonesfun.github.io/notey/notey.html)

Features:
- Save, edit, delete
- Copy any note to your clipboard
- Search
- Tagging
- Export/Import (CSV)
- Optional auto-export
- Character limit, or override
